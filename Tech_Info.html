This website was written using basic HTML and CSS, the basic building blocks for web development. The interactive score examples used in this thesis were encoded using MEI. MEI, or “Music Encoding Initiative,” is an encoding format that is designed to make music machine-readable. It is written according to the MEI schema, “a core set of rules for recording physical and intellectual characteristics of music notation documents expressed as an eXtensible Markup Language (XML) schema.”29

Fig. 3: Unrendered MEI


MEI, in and of itself, is simply code, as seen above. In order to express the MEI as a graphical score like traditional printed music, it must be rendered in Verovio. Verovio is a music notation engraving library built in C++, developed by the Swiss RISM office, which can engrave MEI scores into SVG (for display) or MIDI (for playback) files. One reason that it is particularly well-suited for this task is that it very closely mimics MEI’s internal structure, allowing the user to directly reference elements in an MEI file.30

It is also easily implementable in webpages, such as this one. There are two ways to do this: the app and the Javascript toolkit. The former is more user-friendly, with a GUI complete with built-in buttons for the navigation and zoom functions, but because it currently lacks compatibility with MEI’s editorial markup capabilities, the toolkit has been used here. The navigation and zoom can be coded in the HTML, as they have been for this thesis, allowing access to the full gamut of Verovio’s capabilities. To express information found in MEI’s critical apparatus, elements can be targeted using an xPath query and highlighted however desired in the webpage.

Fig. 4: Verovio toolkit code


The potential pedagogical use cases for this technology are limitless. Though it is used here for a specific philological purpose, other projects such as Unlocking Musicology’s Lohengrin TimeMachine Digital Companion, created by the Oxford e-Research Centre, use it to a more general end (in that case, to illustrate the appearances of a particular leitmotif and its changes over the course of the opera, accompanied by an essay by musicologist Laurence Dreyfus).31
